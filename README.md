# WiFi Auto Login Extension

A sleek and powerful browser extension that allows you to save your captive portal WiFi credentials and log in automatically with a single click, without ever needing to look at the login page.

## Installation

Before you install, ensure you have placed your desired logo image named exactly `logo.png` inside this extension folder.

### Google Chrome
1. Open Google Chrome and navigate to `chrome://extensions/` in the address bar.
2. Enable **Developer mode** using the toggle switch in the top right corner.
3. Click the **Load unpacked** button that appears in the top left.
4. Browse to and select this folder (`AutoLoginExtension`).
5. The extension will now appear in your browser. (We recommend clicking the puzzle piece icon in Chrome and "pinning" it to your toolbar for easy access).

### Microsoft Edge
1. Open Microsoft Edge and navigate to `edge://extensions/` in the address bar.
2. Enable **Developer mode** using the toggle switch in the bottom left corner (or on the left sidebar).
3. Click the **Load unpacked** button at the top right.
4. Browse to and select this folder (`AutoLoginExtension`).
5. The extension is now installed. Make sure it is visible in your toolbar.

## How to Use

1. Click the extension icon in your browser toolbar.
2. Click **Manage Credentials** to open the settings.
3. Enter the URL of the WiFi login page (e.g., `192.168.1.1` or `wifi.college.edu`).
4. Enter your Username/Roll No and Password, then click **Save Settings**.
5. To log in to the WiFi moving forward, simply click the extension icon and select **Login Directly**. The extension will open the login page in the background, fill in your details, submit them, and close the background tab automatically!
